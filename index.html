ASReview IRR plug-in

  • An extension for ASReview that creates automatic reports on the IRR

The data pipeline

  1. Extract relevant information from two ASReview output files
  2. Calculates many statistics of agreement and outputs these as a csv
  3. Outputs a rater summary, a visual summary, and a table

Is this useable?

  • Yes! The set-up is extendible and the R Markdown script is clear

    • Note: Missing data is currently assumed automatically rated as irrelevant!
